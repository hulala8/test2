# 西北大区销售看板 (Mobile Optimized)

这是一个为西北大区销售团队设计的互动数据展示看板。

## 主要功能

- **多维数据分析**：支持按月份、省区、办事处、产品大类和名称进行筛选。
- **KPI 核心指标**：自动计算进货达成率、纯销达成率、同比增长 (YoY) 及环比增长 (MoM)。
- **移动端适配**：
  - **响应式设计**：完美适配 iPhone 与 Android 手机浏览器。
  - **可折叠筛选面板**：针对小屏幕优化的交互。
  - **冻结列滚动**：表格支持横向滚动，第一列（省区/产品）保持固定，方便数据对比。
- **纯前端实现**：无需后端，支持直接导入 CSV/Excel 文件进行数据更新。

## 使用说明

1. 直接在浏览器中打开 `index.html`。
2. 点击右上角的“导入数据源”，上传符合格式要求的 CSV 或 Excel 文件。
3. 即可实时查看各项销售数据的统计结果。

## 开发与适配

本项目基于原有的桌面端看板进行了深度移动端优化，移除了不适合手机展示的图表部分，通过 CSS Sticky 实现冻结列，保证了在大数据量表格下的阅读体验。
